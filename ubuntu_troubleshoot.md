* Switch to console 
```
Ctrl + alt + F1
```

See which processes are running
```
ps -ef
```
If you see a process which has high CPU usage, you can kill it.

```
sudo kill <pid>
```

Of course, you can simply restart the PC too:
```
sudo reboot
```
